# Changelog

All notable changes to `@inharness-ai/c4s-plugin-simple-database-tables` are documented here. Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/); versioning follows [SemVer](https://semver.org/).

## [0.1.0] — 2026-06-26

### Added
- Initial release of the claude4spec plugin contributing the `database-table` entity (SQLite table `database_table`).
- Entity model with columns, indexes, and soft foreign keys.
- Exposure of the entity over MCP, HTTP, and React renderers.
- Plugin manifest targeting host API `^1.0.0` (Node `>=20`).
